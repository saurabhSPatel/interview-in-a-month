# EngineeringOS Project Context

## Project Overview
EngineeringOS is a comprehensive learning platform for software engineering concepts, featuring pattern-based learning across multiple domains including DSA, System Design, and specialized technical topics.

## Mission
To provide senior-level, pattern-first education that enables engineers to understand and implement complex systems without external dependencies.

## Key Principles
- **Pattern-First**: Focus on reusable patterns rather than one-off solutions
- **Standalone Learning**: No external references required
- **Senior-Level Depth**: Target experienced engineers seeking mastery
- **Modern Aesthetics**: Clean, professional design that doesn't look AI-generated

## Technology Stack
- **Frontend**: HTML/CSS/JavaScript (Vanilla)
- **Fonts**: Syne, JetBrains Mono, Inter
- **Hosting**: Static site generation
- **Version Control**: Git
