# Interview-in-a-month

This repository contains a comprehensive 30-day curriculum designed to take a software engineer from foundational concepts to production-ready expertise in Rust, Computer Science fundamentals, and Data Structures & Algorithms (DSA).

### Study Tracks
1.  **Rust — Basics to Advanced**: Focuses on memory safety, zero-cost abstractions, and the unique ownership model. Study this to understand how to build high-performance, safe systems.
2.  **CS Fundamentals**: Covers the "hard" theory of distributed systems, networking, and probabilistic data structures. Essential for senior-level engineering interviews and systems design.
3.  **DSA in 30 Days**: A pattern-based approach to algorithm practice using Go. Instead of grinding random problems, master the underlying patterns (Sliding Window, Monotonic Stack, etc.).

### How to Study
*   **Daily Rhythm**: Spend 1 hour on Rust/CS theory and 2 hours on DSA patterns.
*   **Active Recall**: Use the quizzes provided at the end of each module in the HTML files.
*   **No External Resources**: These pages have been updated to include the necessary context, implementation details, and mental models required to understand the topics deeply.
